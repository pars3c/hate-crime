# hate-crime

## Instructions

  Clone the project
    `git clone https://github.com/pars3c/hate-crime/`
  Go to project directory
    `cd hate-crime`
  Initialize your virtualenv and install the packages from the `requirements.txt`
    `pip install -r requirements.txt`
  Finally, start Jupyter Lab
    `jupyter lab`
